# 你好，世界：从这里开始

欢迎来到你的新博客！这是一个 **纯静态** 的博客模板：不用构建工具、无需数据库，把文件直接丢进 GitHub 仓库就能上线。

## 这个模板有什么特点？

- **排版优先**：文章宽度以阅读舒适为目标（接近 65–75 字符宽）。
- **浅/深色模式**：支持系统主题 + 手动切换，并记住你的选择。
- **搜索与标签筛选**：在首页和归档页都可以即时搜索。

> 写作的本质是表达，而优秀的界面应该把注意力还给内容。
